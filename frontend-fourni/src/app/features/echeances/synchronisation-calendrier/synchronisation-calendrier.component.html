Comment m'abonner dans Google Calendar / Outlook ?
  • Google Calendar : bouton "S'abonner" ci-dessus, ou dans Google Calendar → Autres agendas (+) → À partir de l'URL → collez le lien copié.
  • Outlook / Outlook.com : Ajouter un calendrier → S'abonner depuis le web → collez le lien copié.
  • Apple Calendar (macOS/iOS) : le bouton "S'abonner" ouvre directement l'app Calendrier via le lien webcal://.
